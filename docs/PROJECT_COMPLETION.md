# Project Completion Checklist

This file maps `project_spec.md` MVP/final deliverables to implemented components.

| Spec Requirement | Implementation |
|---|---|
| Synthetic SaaS dataset | `src/generate_data.py`, outputs in `data/raw/` |
| Text classification into revenue-loss taxonomy | `src/revenue_leak/text_taxonomy.py` + classification flow in `src/revenue_leak/features.py` |
| Churn and deal-loss analysis | `src/revenue_leak/analytics.py` |
| Churn risk prediction model | `src/revenue_leak/modeling.py` with persisted model artifact |
| Revenue impact quantification by category | `revenue_loss_summary.csv` generated by `src/revenue_leak/pipeline.py` |
| Operational recommendations | `src/revenue_leak/recommendations.py` |
| Interactive analytics dashboard | `app.py` (Streamlit) |
| Processed outputs for inspection | `data/processed/*.csv` |
| End-to-end runnable pipeline | `python -m src.run_pipeline` |
